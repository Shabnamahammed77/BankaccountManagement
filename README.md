# Bank Management System (Python)

## Description
A console-based Bank Management System developed using Python.
The application allows users to create bank accounts, perform deposits
and withdrawals, check balances, and store data permanently.

## Features
- Create new bank accounts
- Prevent duplicate account numbers
- Deposit and withdraw money
- Check account balance
- View all accounts
- Persistent data storage using pickle

## Technologies Used
- Python
- Object-Oriented Programming (OOP)
- File Handling
- Pickle Module

## How to Run
1. Clone the repository
2. Run main.py
3. Choose options from the menu
